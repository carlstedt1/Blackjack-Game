# Blackjack Game
 Simple version of the Blackjack card game.

FEATURES:
Up to 8 players.
Using MAC emojis to visualize cards. 
Hit or pass.
Automatic Blackjack check.
Automatic Bust check.
Split hand feature.
Automated dealer.
Shows win or loss for all hands including split hands.
